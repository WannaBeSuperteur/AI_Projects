## 1. 개요

* ```cifar_10```, ```fashion_mnist```, ```mnist``` 데이터셋 각각에 대해, **전체 데이터 중 일부만을 Hyper-param 최적화 모델의 학습 데이터로 사용**
* Oh-LoRA가 **Optuna 보다 하이퍼파라미터 탐색을 잘 하게** 하는 것이 목표

## 2. 학습 데이터셋 필터링 기준

* 특정 이미지의 모든 픽셀을 **channel-wise 하게** 평균한 픽셀의 **색상, 채도, 명도** 값의 범위
  * Grayscale 데이터셋의 경우 **명도 값** 의 범위
* 특정 이미지의 모든 픽셀에 대한 **R, G, B 값 각각의 표준편차** 범위
  * Grayscale 데이터셋의 경우 **단일 값의 표준편차** 범위

참고 사항

* class 는 모든 클래스 (10개) 중 **데이터 개수가 가장 많은 5개에 해당하는 sub-dataset** 만 학습/테스트 실시
  * 예: 각 클래스 별 데이터 개수가 ```[10, 150, 250, 20, 5, 75, 500, 60, 300, 400]``` 인 경우, 데이터 개수가 ```150``` ```250``` ```500``` ```300``` ```400``` 인 5개 class 만으로 학습/테스트 실시
* 최대 **70 epochs** 동안 학습 실시
* 다음 조건 만족 시에만 Hyper-param 최적화 모델의 학습 데이터로 사용
  * 전체 학습 데이터 2,000 장 미만
  * 5개 label 중 데이터가 가장 적은 label이 학습 데이터 100장 이상 & 테스트 데이터 20장 이상 (모두 만족)

## 3. Hyper-param 최적화 모델 학습 데이터셋 제작 방법

* 이미지 분류 모델을 먼저 구현
* 해당 분류 모델에 대해 [학습 데이터셋 필터링 기준](#2-학습-데이터셋-필터링-기준) 에 따라 필터링한 여러 개의 랜덤한 sub-dataset 을 생성하여 학습
* 학습 데이터셋의 입/출력 데이터

| 구분     | 설명                                                                                                                                                                                                                                                                               |
|--------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 입력 데이터 | - Encoding 된 Hidden Representation **각 성분 값의 전체 이미지에 대한 평균 및 표준편차**<br>- 모델 하이퍼파라미터 값 (learning rate, LR scheduler, Optimizer, 각 레이어 별 Dropout rate, Activation Func. 등)<br>- 학습 데이터 전체 개수<br>- 데이터 불균형 정도 (**최종 학습되는 5개의** 각 class 별 이미지 개수 분포 관련 정보)<br>- 각 class 의 학습/테스트됨 여부 |
| 출력 데이터 | - **분류 결과의 Macro F1 Score** 점수<br>- 학습 소요 시간의 경우, **특정 epoch 에서의 미세한 loss 차이에 의한 early stopping 여부** 차이에 따른 편차가 큰 경우가 있으므로 평가에서 제외                                                                                                                                               |
